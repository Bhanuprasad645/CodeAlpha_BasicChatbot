# CodeAlpha Basic Chatbot

## Description

A simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

## Features

* Greeting responses
* Current date and time
* Creator information
* Favorite programming language query
* Help command
* Exit command

## Technologies Used

* Python
* datetime module

## Concepts Used

* Loops
* Conditional Statements
* User Input Handling
* String Methods
* Date and Time Functions
